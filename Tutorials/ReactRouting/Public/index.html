Lesson 2

We are adding routing functionality to our application. For this you have to following the steps:

  1. Add the package react-router by command line command : npm install react-router@2.0.0 --save
  2. Now in App folder create coponents
  3. Include the page components in the app.jsx
  4. Now include the React-router components in app.jsx by the include line var {Route, Router, IndexRoute, hashHistory} = require('react-router');
  5. Router component contains the total navigation or the body page that holds all the children
  6. Route component contains the each page,
    1. path : atribute is used to call the page using the url i.e., "app url + path = path of the page"
    2. component : attribute is used to tell the navigation about the page, the value is given as "component={page name given for require() method}" which is included at the top of the app.jsx file
  7. IndexRoute gives out the default page on url or opening of application


Lesson 2

Creating multiple pages for navigation

  1. Create Main.jsx, Nav.jsx, Weather.jsx, About.jsx in App folder as Main, About, Weahter
    1. Main.jsx : is the body of the page
    2. Nav.jsx : is the nav bar of the page
    3. Weather.jsx, About.jsx are the children pages
  2. Let us make Weather.jsx as default children/opening page and About.jsx page to be opened on button click
  3. Create alias in webpack.config.js
  4. Go to app.jsx include Main.jsx, About.jsx and Weather.jsx
  5. Now add following code in app.jsx
    1. ReactDOM.render( < Router history={hashHistory} > <Route path="/" component={Main}> <Route path="about" component={About}/> <IndexRoute component={Weather} /> </Route> </Router>, document.getElementById('app') );
  6. Now in Main.jsx include the nav.jsx and create a Nav tag, then the code in nav.jsx will be added to main.jsx, we are using nav.jsx to create a nav bar
  7. Now in nav.jsx incliude react-router as var {Link} = require('react-router'); This is taken as var Link = require('react-router').Link;
    1. Now add links to About.jsx and Weather.jsx as
    2. <div> <h2>Nav Component</h2> <Link to="/" >Get Weather</Link> <Link to="/about">About</Link> </div>
    3. The attribute that we used to navigate the link is to="url-match-path"
    4. on clicking on Weather link gives url of app sends to Weather page
    5. on clicking on About link gives url+about sends to about page


Lesson 3

Tutorial for Link Tag

  1. Instead of Link tag we can also use <a href="#/about" > Link to About page </a> this works but this cannot provide few functionalities that a Link tag can provide
  2. in Link tag we can know whether the link is in active/selected mode or unselected mode by using the attribute activeClassName="active"
  3. also we can set a style to a selected link by using attribute activeStyle={{fontWeight: 'bold'}}
  4. Instead of Link we can use IndexLink tag the benfit of IndexLink Tag is that this updates the active state of the link where the Link tag once gets active status it won't change the link back to normat state
