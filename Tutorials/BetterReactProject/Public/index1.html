We dont have to use babel type here we can directly use javascript, instead of jsx we can use javascript to use these we have to following a few steps

  1. For this we need a third party package like webpack install this globally in your system by using -g tag
  2. npm install -g webpack@1.12.13
  3. then install locally and for development purpose as npm install --save-dev webpack@1.12.13
  4. Create a file with any name assume app1.js
  5. write a function with documnet.write('Text to display in the function');
  6. This will be displayed in the window/screen
  7. now in the index.html file write the line
  8. script src="bundle.js"
  9. This bundle.js will be created automatically to make this happen use the following command in command line
  10. webpack ./public/app1.js ./public/bundle.js
